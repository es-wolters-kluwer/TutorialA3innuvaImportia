Requisitos para ejecutar

1 - Tener el SDK de .net core 3.0 instalado
2 - Tener instalado los componentes CLI de .net core -> dotnet tool install --global dotnet-ef
3 - Desde una consola de VS en el directorio \a3innuva.Tutorial.Persistence lanzar el comando -> dotnet ef database update
4 - Modificar la seccion PathConfigurationSettings del appsetting.json para informar la path de salida
